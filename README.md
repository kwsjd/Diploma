## Бэктест стратегии

Добавлен скрипт `backtest.py`, который повторяет логику текущего бота на исторических свечах Binance.

Пример запуска:

```bash
python backtest.py --symbol BTCUSDT --interval 1m --days 7
```

Полезные параметры:
- `--tp` и `--sl` — тейк/стоп в долях.
- `--fee` — комиссия на одну сторону в процентах.
- `--spot` — использовать spot-свечи (по умолчанию фьючерсы).

## Зависимости

Установите зависимости перед запуском:

```bash
python -m pip install -r requirements.txt
```
